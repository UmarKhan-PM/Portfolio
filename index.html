Parkinson's Movement-Disorder Monitoring

Master's Research, NUST · Published in Algorithms (MDPI)

Problem

Detecting and classifying Parkinson's movement disorders from wearable sensor data reliably enough to support real monitoring.

My role

Designed a two-stage detection and classification pipeline — classical ML with manual features alongside a deep learning model inspired by RNN-CNN (HARDenseRNN) — evaluated across three window sizes and two labeling techniques, as Master's thesis research.

Outcome

93.03% recall for anomaly detection and 91.54% accuracy for movement-disorder classification on a public dataset. Published as "Towards Effective Parkinson's Monitoring: Movement Disorder Detection and Symptom Identification Using Wearable Inertial Sensors."

93.03% recall 91.54% classification Published research
