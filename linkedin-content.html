Read the full paper here: https://lnkd.in/dridxbnT — This is an early access version, the complete PDF, HTML, and XML versions will be available soon. Here's a quick peak into what we did: This study proposes a two-stage pipeline for detecting and classifying movement disorders in Parkinson's patients using inertial sensor data. It evaluates classical ML with manual features and a deep learning model (inspired by RNN-CNN) while exploring three window sizes and two labeling techniques. Tested on a public dataset, it achieves 93.03% recall for anomaly detection and 91.54% for movement disorder classification, advancing Parkinson's monitoring.
